# android-o-backgroundlocation
Android O后台保持定位示例

### 前述
- 工程基于Android 定位SDK实现
- [高德官网申请Key](http://lbs.amap.com/dev/#/).
- 阅读[参考手册](http://a.amap.com/lbs/static/unzip/Android_Location_Doc/index.html).

## 功能描述 ##
demo展示了在Android 8.0后台执行限制情况下，后台持续定位服务的示例。
